Sobre o Projeto

O APPFILMES é um projeto full-stack desenvolvido para fornecer uma plataforma completa de gerenciamento e exploração de conteúdo cinematográfico. A aplicação foi construída utilizando Python e o framework web Flask, oferecendo uma API RESTful robusta para consulta de informações sobre filmes e séries.

O projeto integra uma interface web moderna com uma arquitetura de backend escalável, permitindo aos usuários visualizar catálogos, realizar buscas avançadas, avaliar produções e interagir com outros cinéfilos através de comentários e avaliações.

Tecnologias Utilizadas

Python

Linguagem de programação principal utilizada para desenvolver toda a lógica de negócio, processamento de dados e integração com APIs externas.

Flask

Framework web minimalista e flexível do Python usado para criar a API RESTful, gerenciar rotas, requisições HTTP e renderização de templates.

API RESTful

Arquitetura de API REST desenvolvida para fornecer endpoints estruturados que permitem consulta, busca e manipulação de dados sobre filmes e séries de forma eficiente.

Funcionalidades Principais

Busca e Catálogo

Sistema completo de busca por título, gênero, ano de lançamento e elenco. Catálogo organizado com informações detalhadas sobre cada produção.

Avaliações e Comentários

Sistema de avaliação por estrelas e comentários dos usuários, permitindo compartilhar opiniões e descobrir novos títulos baseado em recomendações da comunidade.

API RESTful

Endpoints estruturados seguindo padrões REST para integração com outras aplicações, permitindo acesso programático aos dados de filmes e séries.

Interface Responsiva

Design moderno e responsivo utilizando Bootstrap, garantindo uma experiência otimizada em diferentes dispositivos e tamanhos de tela.

Estrutura da API

Arquitetura

A API foi desenvolvida seguindo os princípios REST, utilizando Flask para criar endpoints que processam requisições HTTP (GET, POST, PUT, DELETE) e retornam dados em formato JSON. A arquitetura modular permite fácil manutenção e expansão de funcionalidades.

Endpoints Principais
  • GET /api/filmes - Lista todos os filmes
  • GET /api/filmes/{id} - Detalhes de um filme específico
  • GET /api/filmes/populares - Filmes mais populares
  • GET /api/series - Lista todas as séries
  • POST /api/avaliacoes - Criar nova avaliação
Tecnologias Complementares
  • Bootstrap 5 - Framework CSS para interface responsiva
  • Jinja2 - Engine de templates do Flask
  • JSON - Formato de dados para comunicação API
  • Blueprint - Organização modular de rotas no Flask
